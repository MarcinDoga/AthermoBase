# Security Policy

## Supported Versions

All versions 2.x.x.x

Versions 2.0.0.0 or later support 2-step verification and passwordless login security.

| Version    | Supported          |
| ---------- | ------------------ |
| >= 2.0.0.0 | :white_check_mark: |
| < 2.0.0.0  | :x:                |
